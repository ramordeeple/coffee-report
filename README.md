# **Coffee Report**

CLI для анализа трат студентов на кофе. Программа объединяет данные из нескольких CSV-файлов и вычисляет медианные траты индивидуально для каждого студента.

## **Требования**

* Python 3.12+  
* [uv](https://docs.astral.sh/uv/)

## **Использование**
Запуск:
```bash
uv run python -m src.main --files data/ --report median_coffee
```

Тесты:
```bash
uv run python -m pytest
```

### **Параметры:**

* --files: Путь к папке с CSV-файлами или путь к конкретному `.csv` (например data/math.csv).  
* --report: Тип отчета (поддерживается median-coffee).


## **Формат входных данных**
Используются `.csv` файлы из data

CSV-файлы должны содержать следующие заголовки:

* student: Имя студента.  
* coffee_spent: Сумма, потраченная на кофе.
